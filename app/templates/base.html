Data Scope

Issues in your data, especially spelling errors and missing values, can easily be obscured among thousands of rows.

This application helps you to detect the mentioned issues by providing you a visual summary of the dataset and some simple statistics. Spelling errors and unexpected values will then show up as outliers on these ditribution plots, thereby informing your data cleanup and transformation procedures. After each cleanup, run this again to check the data.

This tool is meant for internal use only. For a more dedicated tool, consider picking up OpenRefine.
